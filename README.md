# Weather-Pattern-Analysis
# Weather Pattern Analysis Project 🌦️

## 📌 Overview
A comprehensive analysis of Finnish weather data (2006-2016) exploring temperature patterns, humidity trends, and correlations between weather variables.

## 🎯 Key Objectives
- Analyze temperature trends and seasonal patterns
- Detect outliers using IQR method
- Apply Z-score standardization
- Calculate 30-day moving averages
- Create correlation heatmaps

## 📊 Key Findings
- **Strongest correlation**: Temperature vs Humidity (-0.636) - warmer air = lower relative humidity
- **Warmest month**: July (23.0°C average)
- **Coldest month**: January (0.8°C average)
- **Most variable month**: February (range: -21.8°C to 19.9°C!)

## 🛠️ Technologies Used
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook
- Statistical Analysis (IQR, Z-score)

## 📈 Visualizations Included
- Temperature trend with 30-day moving average
- Monthly average temperature bar chart
- Correlation heatmap
- Box plots for outlier detection
- Seasonal pattern analysis
